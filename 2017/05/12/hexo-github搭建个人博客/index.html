Hexo + Github搭建个人博客

Posted on 2017-05-12

受张哥 stormzhang 影响，准备搭建一个自己的个人博客。在经过大家的推荐和自己的筛选，最后决定基于 Hexo 搭建 GitHub 博客，搭建的过程中出现了不少问题，也解决了不少问题，在这里写下来供大家交流学习。

软件下载

Git 下载安装

下载 Git
像是安装常规的软件，默认安装即可。
注意：
path 选择如图所示

Tips:

出于安全考虑，Git 的默认设置是只能在 Git Bash 中才能进行 Git 的相关操作。
按照上图进行的选择，将会使得 Git 安装程序在系统 PATH 中加入 Git 的相关路径。
使得你可以在 CMD 界面下调用 Git，不用打开 Git Bash 了。

Node.js 下载安装

下载 Node.js
一路默认安装即可。
Tips:

执行如下命令行检测 Git 是否安装成功
git --version
执行如下命令行检测 Node.js 是否安装成功
node -v
npm -v
显示相关的版本号证明安装成功

GitHub 设置

GitHub 账号注册

去 GitHub 官网注册一个账号，注意要使用邮箱验证。

GitHub 账号设置

登陆主页后新建一个仓库（new repository），命名以 yourname.github.io 的形式，其它项选填，成功创建。
Tips:

检测是否创建成功：
yourname.github.io 网址可以正常那个打开

Hexo 设置

安装 Hexo

新建空白文件夹，按 Shift +鼠标右键，点击“在此处打开命令行窗口”
分别执行如下两行命令
npm install hexo-cli -g
npm install hexo --save
Tips:

执行如下命令行检测 Hexo 是否安装成功
hexo -v
显示相关的版本号证明安装成功

初始化 Hexo

执行如下两行命令
hexo init
npm install

本地预览

执行如下两行命令
hexo g
hexo s
Tips:

如链接打不开，可执行如下命令更改地址
hexo s -p 5000

本地 Hexo 与 GitHub 建立连接

使用 SSH Keys

生成 SSH Key，命令行如下
ssh-keygen -t rsa -C "yourmaile@mail.com"
其中“yourmail@mail.com”为你的邮箱，回车后输入密码，注意此处输入时没有任何显示。
Tips:

执行如下命令行检测是否成功
ssh -T git@github.com

配置 SSH Keys

打开得到的 id_rsa.pub 文件，复制内容，回到 GitHub 页面。 在 Settings 菜单栏下找到 Deploy Keys 栏，然后点击 Add deploy key 按钮，粘贴复制的内容，保存即可。

配置 Deployment

在 GitHub 页面下，找到 Clone or download 按钮，复制其中 SSH 的值。
打开 Hexo 文件夹，找到 _config.yml 文件并打开，修改如下：
deploy:
     type: git
     repo: git@github.com:yourname/yourname.github.io
     branch: master
其中将复制得到的SSH值替换 git@github.com:yourname/yourname.github.io

提交

执行如下命令行
hexo g
hexo d
Tips:

在执行 hexo d 命令行时，如果出现了 ERROR Deployer not found:git 的问题，可以执行如下命令行解决
npm install hexo-deployer-git –save

大功告成，此时打开 yourname.github.io 就和本地预览效果一样啦！

更改域名*

如果不想更改域名，可以选择忽略这一步，继续使用原有的 GitHub 提供的域名即可。

购买域名

在阿里云上购买，价格大约为一年50元左右，我是注册了一个新的账号加上优惠卷：39元/第一年。

解析域名

阿里云有提供的域名解析服务，具体步骤一直在更新，可参考阿里云提供的帮助文档。

绑定域名

回到 GitHub 页面，在 Settings 菜单栏下找到 Custom domain 添加购买的域名。
这样就可以通过自己购买的域名访问自己的博客了，奖励自己一下！但目前只是建造了一个“毛坯房”，具体的优化还得不断完善。

总结

总的来说基于 Hexo 搭建 GitHub 有三个部分：
1.本地 Hexo 配置
2.GitHub 配置
3.本地 Hexo 和 GitHub 建立连接。
其中本地 Hexo 配置包括 Git、Node.js、 Hexo 的安装，GitHub 配置包括 GitHub 的账号申请及配置，本地 Hexo 和 GitHub 建立连接为通过 SSH Keys 建立连接。通过以上三部分就建立一个博客的雏形，其他的完善在雏形上加工修改就可以了。
